# ProcurBosse V5 — 5.0.0

## Version Info
- **Version**: 5.0.0
- **Electron**: 41.x
- **Node**: 20
- **Target**: Win10/11 x64 + Web
- **Status**: UPGRADE

## What's New in V5
- Advanced reporting with Excel/PDF export
- QR code generation for items and documents
- Offline mode with local SQLite sync queue
- Enhanced IP restriction with geo-blocking
- Bulk import from CSV/Excel for items, suppliers
- Multi-language support (English/Kiswahili)
- Auto-backup to Supabase Storage
- Role permissions matrix editor

## Build Instructions
```bash
# Build V5 EXE
python build_procurbosse.py --version 5.0.0 --all --zip

# Or via GitHub Actions (workflow_dispatch with version_override = 5.0.0)
```

- Windows 10/11 (64-bit)
