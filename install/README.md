# ProcurBosse v5.9 — Installation Guide

## Web (Recommended)
Visit: **https://procurbosse.edgeone.app**

## Windows Desktop App
1. Download the latest `.exe` from [GitHub Releases](https://github.com/huiejorjdsksfn/medi-procure-hub/releases)
2. Run `install-procurbosse.bat` as Administrator
3. The app installs to `C:\ProcurBosse\`

## Batch File Installation
- `install-procurbosse.bat` — Full installation (run as Admin)
- `uninstall-procurbosse.bat` — Remove installation

## System Requirements
- Windows 10/11 (x64) or Windows 7 SP1 (ia32)
- Node.js 18+ (for local dev only)
- Internet connection (cloud-hosted ERP)
